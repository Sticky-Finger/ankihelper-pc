  ---
  Anki划词助手 — 产品需求文档 (PRD)

  1. 产品概述

  一款桌面端工具，监听系统剪贴板中的英文文本，自动分词并提供词典查询，帮助用
  户快速制作 Anki 单词卡片。

  目标用户：使用 Anki
  背单词的英语学习者，特别是需要通过阅读英文材料积累生词的用户。

  2. 技术选型

  | 维度      | 方案                          |
  |---------|-----------------------------|
  | 桌面框架    | Flutter Desktop (Dart)      |
  | 状态管理    | Riverpod                    |
  | 本地存储    | SQLite (sqflite_common_ffi) |
  | 剪贴板监听   | clipboard_watcher           |
  | 词典格式    | AnkiHelper 风格 .txt 纯文本词典    |
  | 发音服务    | 有道词典发音 API                  |
  | 翻译服务    | 百度翻译 API / 有道智云 API         |
  | Anki 通信 | AnkiConnect (HTTP 本地接口)     |

  3. 功能模块划分

  整个应用拆分为以下独立模块，每个模块可独立开发与测试：

  模块 A：主题与基础样式系统

  - 基于 Fluent 2 Design 的亮色/暗色双主题 token 系统
  - 应用全局样式基础（字体、间距、圆角、阴影）
  - 国际化（中文本地化）

  模块 B：标题栏 (TitleBar)

  - 应用标题 + 版本号展示
  - 主题切换按钮（亮色/暗色）
  - 设置按钮、关于按钮、手动输入按钮

  模块 C：剪贴板监控与展示 (ClipboardSection)

  - 系统剪贴板实时监听（clipboard_watcher）
  - 原文展示框
  - 原文翻译展示 + 刷新翻译按钮
  - 翻译服务对接（百度/有道 API）
  - 防抖处理（200-300ms）

  模块 D：分词与单词块 (WordsSection)

  - 英文分词引擎
  - 单词块网格展示（支持单击选中 / Shift+单击多选）
  - 当前选中词组显示
  - 选中状态高亮样式

  模块 E：词典查询与结果列表 (ResultsSection)

  - 本地 .txt 词典导入与索引建立
  - 词典查询引擎
  - 结果列表展示（空条目占位 + 释义条目）
  - 查询超时处理（5s 可配）
  - 手动重试刷新
  - 加载中状态展示

  模块 F：卡片操作 (CardActions)

  - "添加卡片" 功能 → AnkiConnect addNote
  - "预览编辑" 弹窗
  - 卡片字段预填充（单词、音标、发音 URL、释义、例句、例句翻译）
  - 例句中选中词汇高亮（<b> 标签）

  模块 G：状态栏 (StatusBar)

  - 应用状态指示（就绪/忙碌）

  - 词典查询状态

  模块 H：通知系统

  - Toast 消息提示
  - 操作反馈

  - 卡片模板选择

  1. 产品定位（一句话）

  Anki划词助手是一款将剪贴板中的英文句子智能切分为单词块，结合本地词典查询快
  速生成 Anki 单词卡片的桌面效率工具。

  2. 用户故事（核心场景）

  场景1：阅读英文文章时快速积累生词
  - 在浏览器或 PDF 阅读器中看到一段英文，复制句子
  - 应用自动切分单词，选中生词后查询词典
  - 一键添加到 Anki，无需手动编辑卡片

  场景2：批量处理多词查询
  - 句子中有多个不认识的单词
  - 通过 Shift+单击连续选中多个单词形成词组
  - 每个单词分别查询并独立添加卡片
  - 例句字段自动填入原文并高亮选中的单词

  场景3：离线环境下的制卡
  - 在无网络的环境下阅读英文资料
  - 依赖本地导入的 .txt 词典完成单词查询
  - 翻译功能暂不可用，但制卡流程不受影响
